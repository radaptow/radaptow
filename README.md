## 🚀 Mi Stack Tecnológico & Caja de Herramientas

### Backend & Desarrollo Web Moderno (Vanilla First)
* **Lenguajes & Stacks:** PHP 8.x (Especialista en arquitecturas monolíticas y vanilla limpias), JavaScript (Vanilla JS), C (Desarrollo a bajo nivel con ncurses).
* **Bases de Datos:** MariaDB, MySQL (Optimización de consultas mediante PDO nativo).

* **Contenedores & Redes:** Docker, Docker Compose (Entornos aislados y desechables desde el Día 1), Cloudflare Tunnels (Exposición segura de servicios locales).
* **Sistemas Operativos:** Linux Power User (Zorin OS Pro para desarrollo principal, Arch Linux para portabilidad, Alpine Linux para microservicios ultra-ligeros).
* **IA Auto-hospedada:** Integración local de herramientas de productividad con Docker (Ollama, Whisper, Stable Diffusion).
